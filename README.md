# ptune
Обучение языковых моделей библиотекой ruPrompts 

# Задание 1
Подготовить набор данных c колонками: **input_text**, **output_text**, **set**, **fold**. 
###### input_text
Входные данные для обучения языковой модели
```
<P>описание_задачи<P>
<P>описание_мира<P>
<P>описание_возможных_команд<P>
<P>команда_от_человека<P>
```
- Количество \<P\> может быть произвольным (можно провести различные запуски, посмотреть как лучше)
###### output_text
Выходные данные, что должна генерировать модель
```
последовательность_команд_робота
```
###### set
К какой выборке относится пример *train* или *test*
###### fold
Разбиение тренировочной части по фолдам, [кросс-валидация](https://scikit-learn.ru/3-1-cross-validation-evaluating-estimator-performance/)
#### Требования
- *Команда от человека* должна строго соответствовать *последовательности команд робота* и *возможным командам*.
- *Описания мира* и *команды от человека* можно комбинировать для получения большего количества входных данных.
- Примерный начальный объем данных ~200 пар примеров.
- Разделить данные на тренировочные и тестовые, например, 0.8/0.2
- 5 фолдов для кросс-валидации
